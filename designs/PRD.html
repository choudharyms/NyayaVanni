# Project: NyayaVanni - Legal Document Analyzer for Indian Users ## Overview A high-trust, AI-powered tool designed to help non-legal experts in India understand complex legal documents in seconds. ## Visual Style & Branding * **Aesthetic:** Clean, minimal, modern, trustworthy. * **Color Palette:** * Primary: Professional blue (#2563EB) * Success: Green (#10B981) * Warning: Amber (#F59E0B) * Danger: Red (#EF4444) * Neutrals: White, light gray (#F3F4F6), dark gray (#1F2937) * **Typography:** Sans-serif system font, large sizes (16px+ body, 22px+ headings), high contrast. * **UI Elements:** Rounded corners, subtle shadows, card-based layout. ## Core Screen Requirements 1. **Landing Page:** Hero section with "Upload Document" CTA, trust badges ("Verified Legal AI"), and language toggle (EN | हिन्दी). 2. **Upload Screen:** Drag-and-drop zone, file picker, camera scan (mobile), and a clear progress bar. 3. **Processing Screen:** Animated loader with progress steps (Extracting text -> Understanding content -> Generating summary). 4. **Results Dashboard (Main Screen):** * **Summary Card:** Plain language explanation. * **Key Details Card:** Parties involved, important dates, and document type category. * **Legal References Section:** Accordion cards with simple explanations of laws/acts. * **Risk Indicator Card:** Color-coded (Green/Yellow/Red) with specific concerns. * **Suggested Actions:** Interactive cards for "Consult Lawyer", "Prepare Reply", etc. 5. **Chat / Ask AI Section:** Fixed bottom panel for asking specific questions about the document. 6. **Document History:** List view of all analyzed documents with status badges. ## Technical & Accessibility * **Device:** Mobile-first but responsive for desktop. * **Accessibility:** WCAG 2.1 AA compliant, high contrast, screen reader compatible. * **Security:** Prominent security badges ("Bank-level encryption") and privacy assurances.
